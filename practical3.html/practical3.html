Question1

Research for and describe the ways to open developers console in Mozilla Firefox and Safari.

In Mozilla Firefox, you can open the developer console and there are two options:
- Right-click on the web page, select "Check item", then go to the "Console" tab
- Press F12
In Safari to open the developer console in both ways:
Go to Safari - Settings - Advanced and check the box labeled "Show the development menu in the menu bar". As soon as the development menu appears in the menu bar, you need to select "Show Error Console" or "Show Web Inspector" to access the developer tools.

Question2

What way of including internal links into the HTML page you would consider generally preferable: absolute or relative links? Describe use cases for choosing absolute links and choosing relative links.

Usually, using relative links for internal linking on an HTML page is better than absolute links. Relative links are easier to maintain, especially when moving files to different folders or when changing file names. Absolute links are necessary when placing links to external websites or to a specific page in another domain, they themselves are inconvenient (well, as for me)
Examples of using absolute links: When placing links to certain pages in another domain When linking to external websites When creating links to resources that can be cached in a CDN
Usage examples for selecting relative links: When creating links to pages within the same domain While maintaining the flexibility of the file structure and organization When creating links to resources within the same folder or subfolder

Question3

Research and write about image formats used in web-development. What is the difference between JPG, GIF and PNG formats? What is WEBP image type? What is SVG?

- JPG: Lossy compression format (for photos (not suitable for images with sharp edges or text) and other detailed images)
- GIF: lossless compression format (supports animation and transparency). This is the best option for simple images, logos and icons.
- PNG: A lossless compression format that supports transparency and is ideal for images with clear edges, text or illustrations.
- WEBP: An image format developed by Google that provides excellent compression and quality compared to JPG and PNG (it supports transparency and animation)
- SVG: A resolution-independent vector image format that is scalable without loss of quality. Ideal for icons, logos, and illustrations that need to be resized dynamically.
